# zhm-aiyouxi-lab

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
这里存放的页面主要用于技术演示、测试、以及个人学习记录。  
所有页面均为独立 HTML 文件，无需依赖外部服务或数据库，可直接在浏览器中打开运行。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放各个独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── README.md           # 本文件
```

- `index.html` 为仓库的入口页面，提供所有归档页面的索引和链接。
- `pages/` 目录下存放各个独立的 HTML 页面，每个页面均为自包含的完整文件。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容可能包含演示性功能、样式测试、交互实验等。
- 每个页面均保持独立，互不干扰，便于单独查看或部署。
- 部分页面可能引用外部资源（如 CDN 上的 CSS/JS 库），但不会收集用户数据或发送网络请求。

## 维护说明

- 本仓库不定期更新，新增或修改页面会直接提交到主分支。
- 欢迎浏览和测试，但请勿将页面内容用于商业用途。
- 如发现页面无法正常显示或功能异常，可通过 Issues 反馈。
- 本仓库不提供任何形式的服务保证，所有页面按“原样”提供。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可以将整个仓库部署到任意静态托管服务（如 GitHub Pages）进行在线访问。

本仓库内容仅供学习、测试和个人参考使用。  
如需引用或修改，请保留原始出处说明。
